Config mongo

ServerURL, CommonDB, UserModel, Extra, Vendor: "mongodb"

- ServerURL: "mongodb://user:password@ip:port/namedb?authSource="
- CommonDB: namedb,
- UserModel: authSource
- Extra : {
A: ServerURL
B: ServerURL
}

Sử dụng được tất cả các hàm xử lý db

VD:
`A.insertOne: A.test`, // insert document mới vào collection test của db A
`A.findAll: B.product`, // Find tất cả document của collection product của db B
`A.Delete: cart`, // Delete cart của db mặc định có thể ghi main.cart = cart

How to use API Core CARMPUSHVK (Common, Controller, Route, Model, Util, Service, View, Config, Hook, context Param) Functions:

  • C: used for functions defined in linked Controller.
  • A: used for functions included by config (Controller, Route config) from Core Common function.
  • R: used for functions defined in linked Route.
  • M: used for functions defined in linked Model.
  • P: used for object in Param/Req when pipeline functions called.
  • U: used for function in Util.
  • S: used for function in Service.
  • H: used for function in Hook.
  • V: used for function in View.
  • K: used for Config Object.
  • Req: used for Param Input, when Input flow is asterisk *.

Some Common function A (usually used to verify model field).

  • A.autoId(): auto create new Mongo Id obejct.
  • A.autoIncrease: model): auto increase this key value.
  • A.autoHash(length = 8): auto generat hash string with length.
  • A.isIdString(): check if a Mongo ID String, return string Id.
  • A.isString(min = -1, max = -1): check if val is String, max: maximum length, min: minimum length.
  • A.isPath(val): check if val is File/Folder path.
  • A.isDate(): check if val is Date, return Date type.
  • A.isNow(): get current Date Time.
  • A.isTime(): check if val is Time hh:mm:ss or number, 01h01m01s = (01 * 3600) + (01 * 60) + 01 = 3661, return Number, mili-second is point decimal.
  • A.isPhone(): check if val is Phone format +xxxxxxxx (6-14 digit), return String Phone value.
  • A.isEmail(): check if val is Email format, return String value.
  • A.isPassword(pass, MinLength, encode): check if val is Password format, return String value.
  • A.isNumber(min, max): check if val is Decimal number, in range of (min, max) if any side defined, return Number value.
  • A.isInteger(min, max): check if val is Integer number, in range of (min, max) if any side defined, return Integer value.
  • A.isBoolean(): check if val is Boolean format, true or false, return Boolean value.
  • A.isArrayIds(): check if val is Array contain all Id string, return Array if Id String value.
  • A.isArray(): check if val is Array format, return Array value.
  • A.isObject(): check if val is Object format, return Object value.
  • A.isNumberEnum(...arrs): check if val is Number Enumeration, list on the rest parameters, return Number value.
  • A.isStringEnum(...arrs): check if val is String Enumeration, list on the rest parameters, return String value.
  • A.isEnumArray(vals, ...arrs): check if vals contains Number/String Enumeration, list on the rest parameters, return vals value.
  • A.isNumberEnumArray(vals, ...arrs): check if vals contains Number Enumeration, list on the rest parameters, return vals value.
  • A.isStringEnumArray(vals, ...arrs): check if vals contains String Enumeration, list on the rest parameters, return vals value.
  • A.isEnum(...arrs): check if val is Number/String Enumeration, list on the rest parameters, return val value.
  • A.isRegex(regex): check if val matched regex expression, return String value.
  • A.isAlpha(min = -1, max = -1): check if alpha string.
  • A.isAlphaNumeric(min = -1, max = -1): check if alpha string or numeric.
  • A.isHexa(): check if it is hexadecimal number.
  • A.isUserName(): check if it is username.
  • A.isCommitMessage(): check if it is git commit message.
  • A.isGitBranch(): check if it is git name.
  • A.isIP(): check if ip address.
  • A.isMAC(): check if MAC address.
  • A.isURL(): check if is url.
  • A.isDomain(): check if is domain.
  • A.isHost(): check if is Host(domain/ip:port).
  • A.isPort(): check if is port.
  • A.isRandom(min, max, floor = false): Randomize a number from min to max, floor is flag to cut to interger.
  • A.noneCheck(): don't/ignore check.

note: when those function failed checked, it return undefined value.

Adjust Object

    - model: tên model đã định nghĩa (có thể có hoặc không có)
    - obj: tạo ra một obj mới

    VD:

    `A.mixObject: abc: ({
    "name" : "xyz",
    "age" : 22
    })`

    Các key sau khi mixObject sẽ được gán vào body (body.name, body.age). Ta có thể gán vào biến như sau:

    `A.mixObject > temp: abc: ({
    "name" : "xyz",
    "age" : 22
    })`

    Các key sẽ được gán vào biến temp (temp.name, temp.age)

    - TH không có model:
    `A.mixObject:: ({
    "name" : "gitgam",
    "age" : 22
    })`

    Kết quả trả về sẽ giống kết quả của obj gán cho từng key

    - TH có model:
    `A.mixObject: abc : ({
    "name" : "xyz", "age" : 22
    })`

    Phải đúng định dạng của model đã định nghĩa kiểu dữ liệu và key tồn tại. Nếu không tồn tại key trong model sẽ bị triệt tiêu còn key không đúng kiểu dữ liệu sẽ cho kết quả undefined

  • A.selectObject: arrs: keyPath: value: matchKeyName='match': restKeyName='rest' : select object in array/pipeData arrs by keyPath-value, return object.

    - arrs: các obj truyền vào
    - keyPath: Key cần select
    - value: giá trị của key để select
    - matchKeyName: mảng chưa các phần tử phù hợp với select mặc định là match
    - restKeyName: mảng chứa các phần tử không phù hợp
    VD:
